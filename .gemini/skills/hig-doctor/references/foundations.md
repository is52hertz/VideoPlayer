# Visual Foundations for VideoPlayer

## Materials & Vibrancy
- **Glass Effect**: Use `.ultraThinMaterial` for control backgrounds to maintain a native yet modern feel.
- **Vibrancy**: Ensure text and icons remain legible over varying video content using vibrant materials.

## Motion & Animation
- **Fluidity**: Use spring-based animations for a more natural, "liquid" feel.
- **Organic Transitions**: Fades and scales should be smooth and responsive to user input.

## SF Symbols
- Use SF Symbols for consistency across all Apple platforms.
- Match symbol weight and scale to surrounding typography.

## Accessibility
- **VoiceOver**: Provide descriptive labels for all playback controls.
- **Dynamic Type**: Ensure the UI scales gracefully for users with larger text settings.
- **Contrast**: Maintain sufficient contrast for text and controls against video backgrounds.
